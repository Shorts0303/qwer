# 메이드 인 매화: 택티컬 아레나 — 쇼케이스

11×7 헥스 맵에서 두 팀의 AI(`MaehwaAgent` 서브클래스)가 5유닛(방패 / 소총×2 / 지정사수 / 의무병)을 지휘해 거점 점령 게이지·전멸·30라운드 판정으로 승부를 가리는 **결정론적 턴제 전술 AI 대전** 프로젝트의 공개 쇼케이스 사이트다.

## 이 사이트에서 할 수 있는 것

- **튜토리얼** — 유닛 / 사거리 / LoS / 점령 게이지 / 1라운드 제한 등 핵심 규칙을 인터랙티브로 익힌다.
- **단일 대전** — 베이스라인 4종(`simple` / `aggressive` / `defensive` / `demo`)끼리 붙이거나, 자기 `agent.py`를 업로드해 시험 대전. 라이브 중계 모드로 한 페이즈씩 사고 과정을 따라가며 볼 수 있다.
- **토너먼트** — 여러 `.py`를 한 번에 업로드해 스위스 4라운드 + 4강·결승 결선을 자동으로 돌린다.

기본 화면에는 가장 강한 베이스라인인 **DemoAgent** 가 사전 로드돼 있어, 별도 업로드 없이도 어떤 매치인지 바로 감을 잡을 수 있다.

## 어디서 동작하나

브라우저 한 개면 끝. Python 엔진은 Pyodide(WebWorker) 위에서 돌고, 모든 계산은 자기 기기에서 이뤄진다. 업로드한 `.py` 파일은 외부로 전송되지 않는다.

## 자기 AI 만들어 보기

이 쇼케이스는 시연·체험용이고, 자기 AI를 코드로 작성·공개·반복 개선하고 싶다면 학생 템플릿 레포에서 시작하자:

1. 템플릿 레포 상단의 **"Use this template"** → "Create a new repository"
2. 자기 레포의 `agent.py` 만 수정·커밋
3. GitHub Actions가 viewer를 빌드 → 자기 Pages URL에서 자기 AI가 대전하는 모습을 본다

규칙: `documents/02_게임_룰북.md`
API 레퍼런스: `documents/03_에이전트_API_스펙.md`

## 라이선스 / 출처

수업용 교보재로 만들어진 프로젝트다. 코드와 문서는 자유롭게 학습·수업에 사용할 수 있다. 원본 마더 레포의 변경사항이 정기적으로 이 사이트에 반영된다.
